# Between Lights: A Nocturnal Survey

A virtual photo exhibition by **Kerem Güman**.

88 night photographs arranged in a free-scrolling virtual gallery space inspired by the installation practices of Wolfgang Tillmans and Hans-Peter Feldmann. The work explores deterritorialization, queer opacity, alienation, and the experience of living in perpetual transit.

## View the Exhibition

Open `exhibition.html` in any browser (desktop or mobile). No internet connection required.

**Live version:** *https://github.com/kerem-guman/betweenlights*

## How It Works

The exhibition is a single HTML file with zero dependencies — no frameworks, no build tools, no server. Drag to navigate, pinch or scroll to zoom, tap any image to enlarge. The full curatorial essay is accessible via the "Exhibition Text" button.

## Structure

```
exhibition.html      — the exhibition (open this)
_gallery/            — web-optimized images for the gallery
*.jpg                — original full-resolution photographs
```
